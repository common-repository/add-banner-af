== Description ==

This plug-in permitted you to put a widget in sidebar with image.
You can manage dimension, link and the tag Alt of image.
There are no problems with the oldest version of Wordpress.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the directory `add_banner_AF` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Aspect > Widget
4. Use it
